Retrieve a user's entire blacklist.

You can use the howMany and startOffset parameters to control the number of results that this method retrieves, as well as the start index for the search results.

Parameters:
userThe user whose blacklist will be retrieved.
howManyThe number of results to retrieve. The default value is 50.
startOffsetThe start index for the search results. The default value is 1. Important: The index's numbering begins at 1, not 0.
cbQueueThe dispatch queue for the callback.
completeCbCalled after the request is completed. Accepts the following parameters:
  • status: Information about the result of the request.
  • error: Information about the error, or nil if there was not an error.
  • blacklistedUsers: An array of MBUser objects representing blacklisted users.
  • startOffset: The starting index for this group of items within the entire list, or nil if the request did not succeed. Important: The index's numbering begins at 1, not 0.
  • totalPossibleResults: The total number of items that can be retrieved, or nil if the request did not succeed.
