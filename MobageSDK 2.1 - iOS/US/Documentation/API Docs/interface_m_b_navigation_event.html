Detailed Description

Stores information about an in-app navigation event.

Generate these events when a user navigates between screens in an app. You can use this information to plot how users move through the app, how long they spend on a screen, and what screen they navigate to next.

Call report: (MBEventReporter-p) to report the event to the analytics server.
