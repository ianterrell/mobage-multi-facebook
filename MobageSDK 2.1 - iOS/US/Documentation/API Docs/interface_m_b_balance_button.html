Detailed Description

The Balance Button for the Mobage Bank, which displays the user's current balance of the app-specific purchased currency and opens the Bank when tapped.

Call getBalanceButton:withCallbackQueue:onComplete: (MBSocialService-p) to retrieve the Balance Button.
