Detailed Description

Determine whether a string contains words that are clearly profane or offensive.

On the US/worldwide platform, this class checks for English-language profanity.

The list of profane and offensive words may be updated at any time.

Check a text string to determine whether it contains words that are clearly profane or offensive.

Parameters:
textThe string to check for profanity.
cbQueueThe dispatch queue for the callback.
completeCbCalled after the request is completed. Accepts the following parameters:
  • status: Information about the result of the request.
  • error: Information about the error, or nil if there was not an error.
  • textIsValid: Set to YES if the string is valid, or NO if the string contains profane or offensive words.
Called if the string is successfully checked for profanity. The textIsValid parameter is set to YES if the string is valid, or NO if the string contains profane or offensive words. Called if an error occurs while checking a string for profanity. The error parameter contains information about the error.
