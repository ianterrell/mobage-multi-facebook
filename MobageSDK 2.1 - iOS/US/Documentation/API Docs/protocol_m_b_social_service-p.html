Detailed Description

Display components of the Mobage user interface.

The following components are available:

  • The Balance Button, which displays the user's balance of app-specific purchased currency.
  • The Bank, which enables a user to check their balance of app-specific purchased currency or buy additional currency.
  • The Friend Picker, which enables the app to retrieve a user-selected group of the user's friends.
  • The Login Dialog, which enables the user to log into Mobage.
  • The Mobage Community user interface.
  • The User Profile screen, which shows a user's Mobage profile.

Retrieve the Balance Button for the Mobage Bank, which displays the user's current balance of the app-specific purchased currency and opens the Bank.

The Balance Button's minimum height is the largest of the following values:

  • 50 pixels.
  • In landscape mode, 10% of the screen's height.
  • In portrait mode, 6% of the screen's height.

The Balance Button's width must be at least three times its height. For example, if the Balance Button's minimum height is 50 pixels, its minimum width is 150 pixels.

Parameters:
rectThe frame for the button.
cbQueueThe dispatch queue for the callback.
completeCbCalled after the request is completed. Accepts the following parameters:
  • status: Information about the result of the request.
  • error: Information about the error, or nil if there was not an error.
  • button: The Balance Button, or nil if the Balance Button could not be retrieved.

+ (void) openFriendPicker: (NSInteger)  maxFriendsToSelect

(NSArray *)  pickedFriends

Open the Friend Picker, which enables the user to choose a list of their friends, with a maximum number of friends that you specify.

The user can select from their entire list of Mobage friends, or they can select only from friends who have used the current app. The selected user IDs are passed to the callback. If any of the selected friends are not using the current app, they will be invited to do so, and the friends' user IDs are passed to the callback.

Calling this method will pause the app until the user exits the Friend Picker.

Parameters:
maxFriendsToSelectThe maximum number of friends that the user may select. Use the value 0 to allow the user to select an unlimited number of friends.
cbQueueThe dispatch queue for the callback.
completeCbCalled after the request is completed. Accepts the following parameters:
  • status: Information about the result of the request.
  • error: Information about the error, or nil if there was not an error.
  • pickedFriends: A list of friends that were chosen, or nil if the user did not choose any friends.
  • invitedFriends: A list of friends that were invited to try the current app, or nil if no friends were invited.
