Detailed Description

Stores information about a custom, app-specific analytics event.

Use this class to report events that do not fit into any other category. Call report: (MBEventReporter-p) to report the event to the analytics server.

Parameters:

payloadAdditional information about the event, or nil if there is no additional information to provide. Each value must be an NSString.
playerStateInformation about the user's current state, such as the user's level and number of lives remaining, or nil to omit this information. Each value must be an NSString. Note: Do not use this method to report events that primarily affect the user's state. Also, do not include information that is unique to the user, such as a user ID or nickname.
